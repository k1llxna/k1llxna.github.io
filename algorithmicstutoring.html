Coding Tutor

I've been tutoring game development to young students since April 2023. Teaching fundementals on what it takes to create games and becoming apart of the video game industry!

I develop flexible learning classes that includes coding, playtesting, and utilizing students current video game hyperfixation. Offering young coders a fun learning journey for their future!


These are my recent graduates from the 2025 Algorithmics Game Design Class that used Roblox Studios!

My goals in teaching students are:

  • Providing one-on-one feedback.
  • Troubleshooting with comprehensive breakdowns.
  • Walkthrough of key principles of level design, game flow and game design for their current projects.
  • Simple coding teachings to get students into the coding mindset.
  • Reinforcing coding and engine practices.
  • Guide students in creating and publishing their own Roblox games, ensuring the application of coding fundamentals like loops, functions, and event handling.
  • Provided in-depth guidance on debugging, optimization, and game publishing, preparing students for advanced development opportunities.

Roblox Studios Game Design Class

My game design class uses the Roblox Studios platform. This is directed to 8-10 year olds who are interested in learning game development and are old enough to dip their toes into coding.




Roblox Studios uses the Luau scripting language, derived from Lua 5.1. It offers enhanced code clarity, error detections, easy Roblox assetintegration and publishing.
This is a great class to get students started into coding and game development as Roblox Studios draws young developers in with their love and interest of Roblox!

Unity Class

I also teach Unity game development to high school students ages 14-18, focusing on C# scripting, 3D modeling, and the fundamentals of game design.
